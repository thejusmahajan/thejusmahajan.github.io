Simulacrum Data Analysis

A biostatistical analysis of synthetic cancer data (Simulacrum v2.1.0), exploring patient demographics, tumour sites, and staging distributions using R.

Key Findings

Most Common Cancers

The analysis of the tumour dataset reveals that Skin (C44), Breast (C50), and Prostate (C61) cancers are the most prevalent in this cohort.

Insight: This distribution mirrors real-world cancer statistics in the UK, validating the utility of the Simulacrum dataset for testing epidemiological hypotheses.

Stage Distribution

Staging is a critical factor in determining prognosis. The data shows a significant number of cases diagnosed at Stage 1 and Stage 4.

However, a large portion of the data is marked as '?' (Unknown) or 'U' (Unstaged). This highlights a common challenge in real-world health data: missing or incomplete records.

Patient Demographics

The patient cohort shows a roughly even split between males and females. The vital status analysis indicates a significant mortality rate, which is expected in a cancer registry dataset.

How It's Done: R Analysis Walkthrough

Here is a glimpse into the R code used to process the data and generate the visualizations.

1

Loading and Cleaning

We load the CSV data and perform basic cleaning. For the tumour analysis, we extract the main ICD-10 code (first 3 characters) to group cancers by type. 

# Load data
df <- read.csv("data/sim_av_tumour.csv")

# Extract main ICD-10 code (e.g., C50 from C50.9)
df$SITE_MAIN <- substr(df$SITE_ICD10_O2, 1, 3)

# Count occurrences
site_counts <- sort(table(df$SITE_MAIN), decreasing = TRUE)
top_10_sites <- head(site_counts, 10)
2

Visualizing Cancer Types

We use a horizontal bar plot to display the top 10 cancer sites. We also map the codes to human-readable names (e.g., C50 -> Breast) for better interpretation. 

# Scale counts for readability (x 10^5)
counts_scaled <- top_10_sites / 100000

# Create plot
barplot(counts_scaled, 
        main = "Top 10 Cancer Sites (ICD-10)",
        xlab = "Count (x 10^5)",
        names.arg = labels,
        col = "steelblue",
        horiz = TRUE,
        las = 1)
