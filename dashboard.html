DeGIR Clinical Dashboard

Interactive Quality Metrics for the German Interventional Radiology Registry

Built during my internship at HealthTwiSt GmbH using R Shiny, this dashboard visualizes quality metrics from the DeGIR registry serving 300+ German radiology clinics. Explore procedure volumes, complication rates, and clinic-level comparisons interactively.

About This Project

The DeGIR (Deutsche Gesellschaft für Interventionelle Radiologie und minimal-invasive Therapie) registry collects procedural data from interventional radiology departments across Germany. During my internship at HealthTwiSt GmbH, I refactored the data processing pipeline and built this interactive dashboard to replace static quarterly PDF reports.

Key features:

  • Clinic-level filtering and comparison across all registered centers
  • Time series visualization of procedure volumes and complication rates
  • Interactive tables with search, sort, and CSV/Excel export
  • KPI value boxes for at-a-glance quality metrics
  • Responsive layout using bslib's modern Bootstrap 5 components

Note: The data shown in this demo uses simulated values for patient privacy. The real dashboard connects to the live DeGIR registry database.
