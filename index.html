Mendel's Experiment: An Interactive Guide

                           
                       
                                 
                                       
                       

Core Concepts for a Programmer

                       
                               
  • Gene: A variable for a trait (e.g., `pea_shape`).
    •                            
  • Allele: The value of that variable. `'A'` for round (dominant), `'a'` for wrinkled (recessive).
    •                            
  • Genotype: The pair of alleles an organism has (e.g., `['A', 'a']`).
    •                            
  • Phenotype: The visible outcome (e.g., if `'A'` is present, phenotype is "Round").
    •                        
                   
                                                    
                       

Step 1: The P (Parental) Generation

                       

Mendel started with two "purebred" plants. One only ever produced round peas, and the other only ever produced wrinkled peas. This means their genotypes were fixed.

                       
                           
                               
AARound
                               

Parent 1

                           
                           
+
                           
                               
aaWrinkled
                               

Parent 2

                           
                       
                       
                   
                                                    
                       

Result: The F1 (First Filial) Generation

                       

When the purebred parents are crossed, every single offspring receives one allele from each parent (`'A'` from the first, `'a'` from the second). This results in a uniform generation.

                       
                           
AaRound
                           
AaRound
                           
AaRound
                           
AaRound
                       
                       
                           

Biological Insight: The Wrinkled Trait Vanished!

                           

This was Mendel's first major discovery. Even though all F1 plants carry the allele for wrinkled (`a`), they all appear round. The round allele (`A`) is **dominant**.

                       
                       

Programmer's View: The Simulation Logic

                       

A simple nested loop simulates this cross perfectly.

                        
# Define the alleles for each "purebred" parent.
parent_1 = ['A', 'A']
parent_2 = ['a', 'a']

# Create an empty list to store the offspring genotypes.
f1_offspring = []

# For each allele from parent 1...
for allele1 in parent_1:
    # ...combine it with each allele from parent 2.
    for allele2 in parent_2:
        f1_offspring.append(allele1 + allele2)

# Result: f1_offspring is ['Aa', 'Aa', 'Aa', 'Aa']
                       
                   
                                                            
                       

Result: The F2 (Second Filial) Generation

                       

Mendel then crossed two plants from the F1 generation (`Aa` x `Aa`). This is where the "vanished" wrinkled trait made a surprising comeback.

                       
                           
AARound
                           
AaRound
                           
aARound
                           
aaWrinkled
                       
                       
                           

Biological Insight: The Famous 3:1 Ratio

                           

The results weren't random. For every 4 offspring, there were statistically 3 round peas for every 1 wrinkled pea. This predictable ratio proved that inheritance followed specific rules.

                       
                       

Statistical Summary

                       
                           
                               

Genotype Ratio

                               

1 AA : 2 Aa : 1 aa

                           
                           
                               

Phenotype Ratio

                               

3 Round : 1 Wrinkled

                           
                       
                       

Statistical Analysis with Code (Pandas)

                       

While we can see the ratios by eye in this small sample, a tool like Pandas is essential for analyzing larger datasets. It can automatically group and count the results for us.

                        
# First, we need the pandas library.
import pandas as pd

# The results from our simulation.
f2_offspring = ['AA', 'Aa', 'aA', 'aa']

# Create a DataFrame to organize the data.
df = pd.DataFrame({'Genotype': f2_offspring})

# Determine the Phenotype for each row.
df['Phenotype'] = df['Genotype'].apply(
    lambda genotype: 'Round' if 'A' in genotype else 'Wrinkled'
)

# Use .value_counts() to get the final statistics.
genotype_stats = df['Genotype'].value_counts()
phenotype_stats = df['Phenotype'].value_counts()

print(phenotype_stats)
# Output:
# Round     3
# Wrinkled  1
                       

Programmer's View: The Simulation Logic

                         

The exact same logic works, just with different parent inputs.

                        
# The parents are now from the F1 generation.
f1_parent_1 = ['A', 'a']
f1_parent_2 = ['A', 'a']

f2_offspring = []

for allele1 in f1_parent_1:
    for allele2 in f1_parent_2:
        f2_offspring.append(allele1 + allele2)

# Result: f2_offspring is ['AA', 'Aa', 'aA', 'aa']
                       
                   
                                                    
                       

Step 3: Proving the Theory (F3 Generation)

                       

Mendel's genius was his next question: "Are all the round F2 peas genetically the same?" To find out, he self-pollinated plants from the F2 generation. He discovered there were three distinct groups.

                       
                           
                               

Test 1: Self-crossing a purebred round (`AA`) pea

                               
                                   
AA
                                   
?
                                   
AA
                                   
AA
                               
                               

Result: All offspring are purebred round. This group is "true-breeding".

                           
                           
                               

Test 2: Self-crossing a hybrid round (`Aa`) pea

                               
                                   
Aa
                                   
?
                                   
AA
                                   
Aa
                                   
aa
                               
                                 

Result: Produces round AND wrinkled offspring, in the familiar 3:1 ratio. This group was not "true-breeding".

                           
                           
                               

Test 3: Self-crossing a wrinkled (`aa`) pea

                               
                                   
aa
                                   
?
                                   
aa
                                   
aa
                               
                               

Result: All offspring are wrinkled. This group is also "true-breeding".

                           
                       
                       
                           

Conclusion: The Law of Segregation

                           

This was the proof! The 3:1 phenotype ratio in the F2 generation was hiding a more fundamental **1:2:1 genotype ratio** (1 `AA` : 2 `Aa` : 1 `aa`). This showed that the two alleles for a trait (`A` and `a`) are discrete units. They don't blend. They separate—or **segregate**—from each other during the formation of reproductive cells, and then recombine in the offspring. This is the cornerstone of modern genetics.
