Berlin's New Growth: Recent Plantings

A data-driven analysis of new tree plantings (2020-2025) in the Steglitz-Zehlendorf district, highlighting modern urban forestry trends and species selection for climate resilience.

Key Findings

A Cohort of Saplings

The dataset specifically tracks newly planted trees, with an average age of just 3.6 years. This is not a survey of the district's ancient oaks and lindens, but rather a receipt of recent greening efforts.

Insight: This data reveals the "nursery age" of trees when they are introduced to the city. It tracks the wave of new life being added to Steglitz-Zehlendorf's established canopy.

Modern Species Selection

This analysis highlights the current choices of urban planners. Unlike the historical dominance of large Planes, the new plantings favor resilient, drought-tolerant species:

  • Tilia cordata 'Greenspire' (Small-leaved Lime)
  • Prunus avium 'Plena' (Double-flowered Wild Cherry)
  • Tilia tomentosa 'Brabant' (Silver Lime)

These species are selected for their ability to survive in warming urban heat islands and limited soil volumes.

Focus of New Plantings

Zehlendorf has received the highest volume of new trees in this dataset (368 saplings). This indicates a specific planting initiative in this locality, rather than it being the "greenest" overall (a title it likely holds, but which requires full inventory data to prove).

How It's Done: A Step-by-Step Guide

Want to perform this analysis yourself? Here is a walkthrough of the Python code used to generate these insights, using pandas for data manipulation and seaborn for visualization.

1

Setting Up the Environment

First, we need to import the necessary libraries. pandas is the go-to tool for data analysis in Python, and seaborn makes creating beautiful plots easy. 

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Define file paths
file_path = 'data/2025_trees_steglitz.csv'
2

Loading and Cleaning Data

We load the CSV file. Since German CSVs often use semicolons ; as separators, we specify that. We also calculate the age of the trees by subtracting the planting year (pflanzjahr) from the current year (2025). 

# Load Data
print("Loading data...")
df = pd.read_csv(file_path, sep=';')

# Data Cleaning
# Convert planting year to numeric, handling errors
df['pflanzjahr'] = pd.to_numeric(df['pflanzjahr'], errors='coerce')

# Filter for valid years (e.g., between 1700 and 2025)
df = df[(df['pflanzjahr'] >= 1700) & (df['pflanzjahr'] <= 2025)]

# Calculate Age
df['age'] = 2025 - df['pflanzjahr']

print(f"Data loaded: {len(df)} trees.")
3

Visualizing Species Diversity

To find the most common trees, we count the unique values in the baumart (tree species) column and take the top 10. Then we plot them using a bar chart. 

# Get top 10 species
top_species = df['baumart'].value_counts().head(10)

# Create Plot
plt.figure(figsize=(12, 6))
sns.barplot(x=top_species.values, y=top_species.index, palette='viridis')
plt.title('Top 10 Tree Species')
plt.xlabel('Count')
plt.tight_layout()
plt.show()
4

Visualizing Age Distribution

A histogram is perfect for showing the distribution of planting years. This helps us see "waves" of planting activity. 

plt.figure(figsize=(10, 6))
sns.histplot(df['pflanzjahr'], bins=20, kde=True, color='green')
plt.title('Distribution of Planting Years')
plt.xlabel('Year')
plt.ylabel('Count')
plt.tight_layout()
plt.show()
