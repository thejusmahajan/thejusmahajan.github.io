Applied Biostatistics: PCA and Survival Analysis in R

A deep dive into clinical biostatistics. My study notes on implementing Principal Component Analysis for high-dimensional data reduction and Kaplan-Meier estimators for clinical survival analysis.

1. Principal Component Analysis (PCA) for Genomics

When dealing with high-throughput sequencing data (like RNA-seq or single-cell datasets), we frequently encounter the "curse of dimensionality" — having tens of thousands of genes (features) for only hundreds of patients (observations).

Principal Component Analysis (PCA) is our primary weapon for dimensionality reduction. It linearly transforms the data into new orthogonal variables (Principal Components) that capture the maximum variance.

Implementation with prcomp()

While packages like DESeq2 have built-in plotting functions (plotPCA()), it's crucial to understand how to do it natively to maintain flexibility:

# Assuming 'expr_matrix' holds log-normalized gene expression (genes as columns, patients as rows)
# Center and scale the data (scaling is critical if features have different units, though less so for strictly normalized RNA-seq)
pca_result <- prcomp(expr_matrix, center = TRUE, scale. = TRUE)

# The rotation matrix contains the gene loadings
head(pca_result$rotation[, 1:2])

# Plotting the variance explained (Scree Plot)
variances <- pca_result$sdev^2
pve <- variances / sum(variances)
plot(pve, xlab="Principal Component", ylab="Proportion of Variance Explained", type="b")

2. Survival Analysis: The Kaplan-Meier Estimator

In clinical biostatistics, we rarely just want to know *if* an event (like disease progression or death) occurred; we want to model *when* it occurred, while accounting for right-censored data (patients who either dropped out of the study or didn't experience the event by the study's end).

Modeling Survival Curves

The survival package in R is the gold standard for this. The Kaplan-Meier estimator provides a non-parametric statistic used to estimate the survival function from lifetime data.

library(survival)
library(survminer) # For beautiful ggplot2-based survival plots

# Create the survival object: time to event and event status (1=censored, 2=event)
surv_obj <- Surv(time = clinical_data$overall_survival_months, 
                 event = clinical_data$vital_status)

# Fit the Kaplan-Meier curve, stratifying by a biomarker (e.g., TP53 mutation status)
km_fit <- survfit(surv_obj ~ TP53_Status, data = clinical_data)

# Visualize
ggsurvplot(km_fit, 
           data = clinical_data,
           pval = TRUE, # Automatically calculates log-rank p-value
           conf.int = TRUE,
           risk.table = TRUE,
           palette = c("#E7B800", "#2E9FDF"),
           title = "Overall Survival by TP53 Mutation Status")

Key Takeaway for Memory

Always remember: the Log-rank test (which ggsurvplot(pval=TRUE) outputs) tests the null hypothesis that there is no difference between the population survival curves. If our p-value < 0.05, our biomarker is statistically significant in predicting survival differences.
