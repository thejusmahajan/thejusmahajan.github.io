# Dr. Thejus Mahajan - Personal Portfolio & Website

Source code for my personal portfolio website and blog, developed to showcase my projects in bioinformatics, pipeline engineering, and data science.

## Tech Stack
- **HTML5 & CSS3**
- **Tailwind CSS**
- **Vanilla JavaScript**
- **Prism.js**

## Local Development
As a purely static site, you can run this locally using any simple web server. For example:

```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.
